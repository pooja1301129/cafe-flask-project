# ☕ Cafe Management System (Flask Web App)
A simple beginner-friendly Cafe Management System built using Python Flask, HTML, and CSS.  
This project simulates a basic restaurant ordering system where users can enter food items and get the total bill calculated using Python backend logic.
---
## 🚀 Features
- Display cafe menu
- Accept multiple food items from user
- Calculate total bill using Python (Flask backend)
- Show ordered items
- Display final bill dynamically
- Simple and clean UI using HTML & CSS
---
## 🧰 Tech Stack
- Python 🐍
- Flask (Web Framework)
- HTML (Structure)
- CSS (Styling)
